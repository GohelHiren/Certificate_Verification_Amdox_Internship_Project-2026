# Certificate_Verification_Amdox_Internship_Project-2026


## Description
This project is a internship Certificate Verification System web application developed as part of the Amdox Technologies Internship Project 2025. The application allows users to search a certificate and Download, while Authority person can submit certificate data for all intern student .

## How to Run the Application
1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   ```
2. **Navigate to the Project Directory**:
   ```bash
    cd Certificate_Verification_Amdox_Internship_Project-2026
    ```
3. **frontend Setup**:
   - Navigate to the frontend directory:
     ```bash
     cd frontend
     ```
   - Install dependencies:
     ```bash
     npm install
     ```
   - Start the frontend server:
     ```bash
     npm run dev
     ```
4. **backend Setup**:

    - Open a new terminal and navigate to the backend directory:
      ```bash
      cd backend
      ```
    - Install dependencies:
      ```bash
      npm install
      ```
    - Start the backend server:
      ```bash
      npm start
      ```
## Technology Stack
The application is built using the MERN stack, which includes:

- **MongoDB**: A NoSQL database for storing intern student certificate data and user information.
- **Express.js**: A web application framework for Node.js, used to build the backend API.
- **React.js**: A JavaScript library for building user interfaces, used for the frontend.
- **Node.js**: A JavaScript runtime for building the backend server.

## Features
- User registration and authentication
- Certificate search and download
- Admin upload Intern student data
- Responsive design for mobile and desktop
